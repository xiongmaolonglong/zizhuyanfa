# 超级管理后台实现计划

> **面向 AI 代理的工作者：** 使用 superpowers:subagent-driven-development 逐任务实现。

**目标：** 构建平台方专用的监管后台，可查看所有租户数据、全局统计、租户管理

**架构：** Vue3 + Vite + Element Plus，API 使用 `/api/v1/admin/` 前缀（无 tenant_id 过滤）

**技术栈：** Vue 3, Element Plus, ECharts, Pinia, Axios

---

## 项目初始化

- [ ] **步骤 1：搭建项目**

```bash
npm create vite@latest super-admin -- --template vue
cd super-admin
npm install vue-router pinia axios element-plus @element-plus/icons-vue echarts
```

目录结构同 admin-web，页面放在 `src/views/` 下

- [ ] 步骤 2：配置 + 登录页（平台管理员专用登录）
- [ ] 步骤 3：布局框架（侧边栏菜单：工作台、租户管理、工单穿透、甲方监管、系统配置）
- [ ] Commit

- 该租户的部门与人员
