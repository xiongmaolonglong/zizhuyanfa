# 甲方小程序实现计划

> **面向 AI 代理的工作者：** 使用微信原生开发。小程序 = 业务员用，现场拍照申报、查看进度、接收通知。
> 甲方 Web 端 = 领导/管理员用，审批和管理。

**目标：** 甲方业务员通过微信小程序快速申报、查看进度、接收通知

**技术栈：** 微信原生 WXML + WXSS + JS，调用后端 API（不走云开发）

## 页面 1：登录

**布局：**
```
┌───────────────────────┐
│      广告工程申报       │
│      [Logo]            │
│                        │
│  [微信授权登录]         │
│                        │
│  首次登录自动注册        │
└───────────────────────┘
```

**功能：**
- 点击按钮调用 `wx.login` 获取 code
- 调用后端 `/api/v1/auth/client/login`，传 code + 手机号（`getPhoneNumber`）
- 登录成功存 token 到 storage，跳转首页
- 首次登录自动创建账号（需业务员提前在甲方 Web 端被管理员添加到系统中）

**注意：** 业务员需要先由甲方管理员在 Web 端添加账号（姓名+手机号），小程序登录时匹配手机号。未注册的手机号提示"请联系管理员添加账号"。

```
│  [选择地址]              │
│  湖南省长沙市岳麓区XX街道│
│  详细地址 *             │
│  [输入详细地址]          │
│                        │
│  需求描述 *             │
│  [多行输入，最多500字]    │
│  0/500                  │
│                        │
│  现场照片              │
│  ┌───┐ ┌───┐ ┌───┐    │
│  │ + │ │📷 │ │📷 │    │
│  └───┘ └───┘ └───┘    │
│  最多9张，单张≤10MB      │
│                        │
│  附件（选填）           │

└───────────────────────┘
```

**功能：**
- 表单验证（必填项）
- 项目类型：预设选项（门头招牌/灯箱/LED屏/户外大牌/室内广告/其他），可配置

- 现场照片：调用 `wx.chooseMedia` 拍照或从相册选，压缩后上传

- 提交成功后跳转申报详情

## API 封装

`utils/api.js` - 统一请求封装：
- 自动注入 token
- 401 跳转登录
- 错误 toast 提示
- 基础 URL 可配置

模块 API：
- `api.auth.js` - 登录
- `api.declare.js` - 申报 CRUD
- `api.address.js` - 地址查询
- `api.file.js` - 文件上传
- `api.notification.js` - 消息通知
