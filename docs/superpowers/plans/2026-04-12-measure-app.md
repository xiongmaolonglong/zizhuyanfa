# 测量 APP 实现计划

> **面向 AI 代理的工作者：** Flutter 开发，iOS + Android 一套代码。测量员 + 施工员共用。
> 支持离线缓存、自动同步。

**目标：** 测量/施工人员现场作业工具，支持测量数据采集、施工记录、离线同步、客户签名

**技术栈：** Flutter 3.x, sqflite（离线缓存）, http, image_picker, shared_preferences, signature, connectivity_plus, path_provider

## SQLite 本地数据库

**表结构：**
```sql
-- 待办任务缓存
CREATE TABLE tasks (
  id INTEGER PRIMARY KEY,
  order_no TEXT,
  title TEXT,
  task_type TEXT,  -- 'measurement' 或 'construction'
  address TEXT,
  deadline TEXT,
  cached_at TEXT
);

-- 本地测量草稿
CREATE TABLE measurement_drafts (
  id INTEGER PRIMARY KEY,
  order_id INTEGER,
  basic_info TEXT,    -- JSON
  materials TEXT,     -- JSON
  signature_path TEXT,
  created_at TEXT,
  status TEXT         -- 'draft' / 'submitted' / 'failed'
);

-- 本地施工草稿
CREATE TABLE construction_drafts (
  id INTEGER PRIMARY KEY,
  order_id INTEGER,
  photos TEXT,        -- JSON
  notes TEXT,
  duration TEXT,
  signature_path TEXT,
  created_at TEXT,
  status TEXT
);

-- 同步队列
CREATE TABLE sync_queue (
  id INTEGER PRIMARY KEY,
  type TEXT,          -- 'measurement' / 'construction' / 'exception'
  order_id INTEGER,
  payload TEXT,       -- JSON
  created_at TEXT,
  status TEXT,        -- 'pending' / 'synced' / 'failed'
  error_msg TEXT,
  retry_count INTEGER DEFAULT 0
);

-- 本地图片缓存
CREATE TABLE image_cache (
  id INTEGER PRIMARY KEY,
  order_id INTEGER,
  local_path TEXT,
  remote_url TEXT,
  type TEXT,          -- 'photo' / 'sketch' / 'annotation' / 'signature'
  created_at TEXT
);
```
