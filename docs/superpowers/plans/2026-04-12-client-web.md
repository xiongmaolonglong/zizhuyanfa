# 甲方 Web 端实现计划

> **面向 AI 代理的工作者：** 使用 superpowers:subagent-driven-development 逐任务实现。

**目标：** 构建甲方使用的申报端，支持申报单填报、审批、进度跟踪

**架构：** Vue3 + Vite + Vue Router + Pinia + Element Plus，与 admin-web 共享后端 API

**技术栈：** Vue 3 Composition API, Element Plus, Vue Router, Pinia, Axios

---

## 项目初始化

- [ ] **步骤 1：搭建项目**

```bash
npm create vite@latest client-web -- --template vue
cd client-web
npm install vue-router pinia axios element-plus @element-plus/icons-vue
```

目录结构：
```
client-web/src/
├── api/
├── assets/
├── components/
│   ├── Layout/
│   └── common/
├── router/
├── store/
├── views/
│   ├── login/
│   ├── dashboard/
│   ├── declarations/
│   ├── approvals/
│   └── settings/
├── utils/
├── App.vue
└── main.js
```

vite proxy 配置指向后端 `http://localhost:3000`

- 施工环节：施工记录照片，[验收签字] 按钮（手写签名）
